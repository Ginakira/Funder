# Funder

基于 Qt 5 的跨平台基金信息统计助手

![image.png](https://i.loli.net/2021/07/01/wPYLBxSya1XAgR7.png)

## 支持功能

- 查看净值、估值等基金信息
- 统计收益率、持仓收益等个性化信息
- 兼容 Windows / Mac / Linux 平台

## 致谢

- 本项目参考了萧然独奏于52pojie发布的[发了个财](https://www.52pojie.cn/forum.php?mod=viewthread&tid=1220080)
